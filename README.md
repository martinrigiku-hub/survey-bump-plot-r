# Bump Plot Example in R

This repository contains a reproducible example of a bump plot created in R using dummy survey data.

## What the script does

- Generates dummy data for 9 groups
- Simulates responses to 24 survey items on a 1-5 scale
- Aggregates mean scores by group and item
- Ranks groups within each item
- Visualizes rank shifts across items with a bump chart

## Files

- `bump_plot_dummy.R` – complete R script
- `bump_plot_dummy.png` – exported example figure

This example uses dummy data only and is intended for demonstration purposes.
